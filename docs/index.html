resources

Creates a set of routes that describe the list of resources

resources(name: string, configuration: ResourcesConfig, controller: Object, createChilds: Function?)
Parameters
name (string) Base name in lowercase. From name a path and memberId will be created.
configuration (ResourcesConfig) Setup for resource
controller (Object) Object with handlers
createChilds (Function?) Childs will be added to resource root
Example
resources('article', {}, ArticlesController)
resources('publisher', {}, PublishersController, childs(
  member(
    resources('magazine', {}, MagazinesController, childs(
      member(
        resources('photo', {}, PhotosController),
        resources('article', {}, ArticleController)
      )
    ))
  )
))
